Projeto WorldBeauty - Programação Orientada a Objetos

O projeto WorldBeauty é um trabalho orientado pelo professor Gerson da Penha, da disciplina de Programação Orientada a Objetos da FATEC São José dos Campos. O projeto tem o foco de apresentar programação WEB utilizando tecnologias Java como Spring Boot, Hibernate e Thymeleaf.
